## 一、MVP目标

### 1.1 核心功能 ✅

实现最小可用版本，能够：
1. ✅ 解析 `.h` 头文件，提取函数声明、结构体、枚举、宏定义
2. ✅ 解析 `.c` 源文件，提取函数实现
3. ✅ 进行基础的声明-定义一致性校验
4. ✅ 生成基本的 Markdown API 文档

### 1.2 已实现功能

- ✅ 函数声明/定义解析
- ✅ 结构体解析（支持 typedef struct）
- ✅ 枚举解析（支持 typedef enum + 行尾注释）
- ✅ 宏定义解析（分类：硬件配置/协议常量/状态码）
- ✅ 注释提取（支持 Doxygen 风格 + 章节分隔符）
- ✅ 签名一致性校验
- ✅ Markdown 文档生成

## 四、已知限制

1. **行尾注释格式**: 仅支持 `//` 风格的行尾注释
2. **复杂表达式**: 枚举值中的复杂表达式无法解析
3. **嵌套结构体**: 嵌套结构体解析有限制
4. **条件编译**: 暂不支持条件编译块解析
